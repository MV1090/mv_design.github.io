Crossbow Lady

Engine:   Unreal

Platform:   Windows

Language:   C++

Role:   Game Designer/ Solo Programmer

Summary:
Crossbow Lady a simple first person shooter made entirely using C++
The player must navigate through a simple level, dodge the enemy turrets and destroy all the targets.

Highlights:

  • C++
  • Interactable objects
  • AI Behaviour
  • Pick ups
  • Health Component
