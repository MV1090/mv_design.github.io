Alex Kidd remake

Engine:   Unity

Platform:   Windows

Language:   C#

Role:   Solo designer/ Programmer

Summary:
Alex Kidd is a personalized remake of the 1989 2D platformer, Alex Kidd in the Enchanted Castle.
The player must use some of the classic mechanics from the Alex Kidd series, such as the pogo stick, to avoid enemy's and navigate through a custom designed level.

Highlights:

  • Check point system
  • Inventory system
  • Unity's Input system
  • Animation events
  • Crates with unique destructible patterns
